# 📚 Student List App (Список студентов)

**Краткое описание:**  
Приложение представляет собой список студентов с их именами, возрастом и краткими юмористическими описаниями. Каждый элемент списка отображает аватарку студента, имя, возраст и возможность раскрыть подробное описание. Приложение разработано на Kotlin с использованием Jetpack Compose и демонстрирует применение принципов Material Design 3.

## ⚙️ Функциональность

Приложение реализует следующие возможности:

| № | Функция | Описание |
|---|---------|----------|
| 1 | **Список студентов** | Отображение 10 студентов с уникальными именами и описаниями |
| 2 | **Аватарки** | Круглые изображения студентов (64x64dp) |
| 3 | **Информация о студенте** | Имя (жирный шрифт) и возраст (обычный шрифт) |
| 4 | **Раскрытие описания** | Кнопка ▼/▲ для показа юмористического описания студента |
| 5 | **Анимация** | Плавное раскрытие/сворачивание карточки (Spring Animation) |
| 6 | **Светлая тема** | Кастомная цветовая схема с зелёным акцентом (#006C4C) |
| 7 | **Тёмная тема** | Автоматическая адаптация под системную тёмную тему |
| 8 | **Top App Bar** | Верхняя панель с логотипом и названием приложения |
| 9 | **Карточки (Card)** | Скруглённые углы у карточек (bottomStart, topEnd = 16dp) |
| 10 | **Кастомные шрифты** | Montserrat (основной) и Abril Fatface (заголовки) |
